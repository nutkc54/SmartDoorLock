# SmartDoorLock
Project Description:
A smart door lock system that uses a keypad and password authentication to control access. The system allows users to enter a PIN code to unlock the door. It can send notifications via Telegram when an incorrect password is entered multiple times and activates a buzzer for security alerts.

## 🔧 Hardware

- ESP32  
- Keypad 4x4  
- Servo Motor (for door lock mechanism)  
- Buzzer  
- LCD 16x2 (optional for display)  

## 💻 Software / Libraries

- Arduino IDE  
- WiFi.h  
- HTTPClient.h  
- Keypad.h  
- Servo.h  
- LiquidCrystal_I2C.h
